# User Stories

## Template
Sebagai [jenis pengguna], saya ingin [tujuan/aksi], sehingga [manfaat].

## Daftar User Story

### US01 - Membuat akun pelanggan
Sebagai admin, saya ingin membuat akun pelanggan baru, sehingga data pelanggan dapat tersimpan di sistem.

### US02 - Melihat daftar akun pelanggan
Sebagai admin, saya ingin melihat daftar akun pelanggan, sehingga saya dapat memantau seluruh data pelanggan.

### US03 - Melihat detail akun pelanggan
Sebagai admin, saya ingin melihat detail akun pelanggan, sehingga saya dapat mengetahui informasi lengkap pelanggan tertentu.

### US04 - Memperbarui akun pelanggan
Sebagai admin, saya ingin memperbarui data akun pelanggan, sehingga informasi pelanggan tetap akurat.

### US05 - Menghapus akun pelanggan
Sebagai admin, saya ingin menghapus akun pelanggan, sehingga data yang tidak lagi diperlukan dapat dibersihkan.

### US06 - Validasi data pelanggan
Sebagai sistem, saya ingin memvalidasi data input pelanggan, sehingga data yang tersimpan konsisten dan benar.

### US07 - Pencarian pelanggan
Sebagai admin, saya ingin mencari pelanggan berdasarkan nama atau email, sehingga saya dapat menemukan data dengan cepat.

### US08 - Health check service
Sebagai operator, saya ingin endpoint health check tersedia, sehingga saya dapat memantau status layanan.
