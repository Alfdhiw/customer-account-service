# Customer Account Service

Microservice untuk mengelola akun pelanggan dalam proyek capstone.

## Deskripsi
Layanan ini menyediakan API untuk membuat, melihat, memperbarui, dan menghapus data akun pelanggan.

## Tujuan Proyek
- Menerapkan praktik pengembangan perangkat lunak modern
- Menggunakan GitHub untuk manajemen proyek
- Menyiapkan CI/CD
- Mengontainerkan aplikasi dengan Docker
- Menyebarkan aplikasi ke Kubernetes

## Tech Stack (rencana)
- Backend: bebas sesuai instruksi kampus
- Testing: otomatis
- CI/CD: GitHub Actions
- Container: Docker
- Deployment: Kubernetes
